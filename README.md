# Automated Malignancy Grading in Histopathological Images Using Explainable X-CNNs

## Overview

An AI-powered web application for automated malignancy grading of histopathological images using Explainable Convolutional Neural Networks (X-CNNs). The system classifies tissue images and provides visual explanations to support clinical decision-making.

## Features

* Histopathological image classification
* Breast cancer and prostate cancer analysis
* Explainable AI (XAI) visualization
* Interactive React frontend
* Python backend for inference
* Deep learning model integration using PyTorch
